Privacy Policy

Effective Date: 20 March 2025

1. Data Collection and Usage

This extension collects and uses user data for the following purposes:

  • Geolocation access is used to display accurate prayer timings based on the user's current location.
  • Top sites permission is used to display the most visited sites when a new tab is opened for easy access.
  • Local storage is used to store and display Hadiths and Quizzes.
  • Favicon usage is for aesthetic purposes only.
  • External API access (Al-Adhan) is used to fetch prayer times and related data.
  • Remote files (data.json) are used to fetch and display Tasbih (Arabic) content.

2. Data Sharing

The collected data is not shared with any third parties except for the Al-Adhan API, which is used to fetch prayer times.

3. Data Security

We take reasonable measures to protect your data. However, we do not store or share any personal information beyond what is required for the functionality of the extension.

4. User Control

Users can disable location access, local storage, or remove the extension at any time if they do not wish to share data.

5. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted here.

6. Contact Information

If you have any questions about this policy, please contact us at deenipedia@gmail.com.
